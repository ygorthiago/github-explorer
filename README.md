## 🎲 Rodando o projeto

- ### Clone or download this repositoy. If you want to clone:
  ```bash
  $ git clone https://github.com/ygorthiago/github-explorer.git
  ```

- ### Access project folder on command prompt and install the dependencies:

  ```bash
  $ npm install

  or

  $ yarn install
  ```

- ### Execute "start" script:
  ```bash
  $ npm start

  or

  $ yarn start

  # the project will start on port: 3000 - access http://localhost:3000
  ```

  ### Okay! You're ready to access the application and explore GitHub's repositories.
